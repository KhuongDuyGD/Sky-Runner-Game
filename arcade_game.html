🚀 Version 12.0 - "Visual and Game Feel Big Update"

  • 🏔️ Procedural Parallax Backgrounds: 3-layer scrolling system (sky/stars/moon, mountains, ground) replacing static backgrounds
  • 🌙 Dynamic Moon & Stars: Radial-gradient moon (cyberpunk = magenta, horror = blood red, dark = pale) with twinkling stars using Math.sin alpha animation on dark themes
  • ⛰️ Procedural Mountain Ranges: Two ranges of noise-based mountain silhouettes scrolling at different parallax speeds (15% far, 35% near)
  • 🌿 Animated Ground Layer: Swaying grass blades, moving detail lines, and depth striping replacing the old static ground
  • 💡 Additive Blending (Neon Glow): Cyberpunk/Glitch/Rainbow themes use globalCompositeOperation = 'lighter' for neon glow effects on all game objects
  • 🎀 Player Ribbon Trail: 15-point position history drawn as a fading polyline behind the player, with neon glow on cyberpunk/glitch styles
  • 💫 Projectile Energy Trail: Outer glow ring + 3 fading energy circles behind each projectile with additive blending
  • 🔲 Obstacle Neon Glow: Outer neon glow rect + animating energy scan line on obstacles for cyberpunk/glitch themes
  • 🦇 Aerial Enemy Living Geometry: Sinusoidal Y-axis bobbing with per-instance random speed and amplitude, pulsating aura, and energy ring effects
  • 💀 Trap Obstacle VFX: Bobbing Y motion, pulsating outer glow rings, and rotating inner elements
  • 🐉 Boss Attack VFX: Laser outer glow + bright inner core; projectile outer glow + hot center; additive blending on dark themes
  • 📳 Screen Shake System: Camera shake with decaying intensity on death (12), projectile kills (6), trap hits (5), shield blocks (4), and boss kills (15)
  • ⏸️ Hit-Stop System: 4-frame physics freeze on projectile obstacle destruction while rendering continues — classic "game feel" technique
  • 🌈 Chromatic Aberration: Red/cyan color offset post-processing effect — persistent on Glitch theme, burst on death and boss hit
  • 🌧️ Rain Ground Splash: Rain particles spawn 3 tiny upward-bouncing splash particles on ground impact
  • ⛈️ Storm Neon Rain: Additive blending on rain during storms for cyberpunk/glitch/dark themes
  • 🖼️ Enhanced Border: Decorative corner brackets with glitch wobble effect for the Glitch theme
  • 🔧 BackgroundManager Class: New dedicated class managing all parallax layers, scrolling, and procedural generation
  • 🔧 FXManager System: New dedicated system for screen shake, hit-stop, and chromatic aberration with full reset support
  • 🎮 Zero External Assets: All visuals are 100% procedural using Canvas API — no images, no sprites, no external files
